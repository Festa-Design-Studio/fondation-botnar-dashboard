Accessibility Guidelines

Focus Management

  • Focus trap: Focus is contained within the modal when open
  • Initial focus: First interactive element receives focus on open
  • Return focus: Focus returns to trigger element on close
  • Tab order: Logical tab sequence within modal content

ARIA Implementation

  • role="dialog" for content modals and confirmations
  • role="alertdialog" for alert notifications
  • aria-labelledby pointing to modal title
  • aria-describedby for additional context

Keyboard Navigation

  • Escape key closes modal
  • Enter key activates primary action
  • Tab/Shift+Tab for navigation within modal
  • Arrow keys for drawer panel positioning
