Best Practices

✓
Use for primary actions
Reserve primary buttons for the main action on a page or section
✓
Keep text concise
Use clear, action-oriented text that describes what will happen
✗
Don't use multiple primary buttons
Only one primary button should be visible in a given context
✗
Don't disable without explanation
Always provide context when a button is disabled
